# Sistema de Gestión – Estudio de Abogados

## Descripción
Proyecto académico para el desarrollo de un sistema de gestión que permita administrar clientes y expedientes legales de un estudio de abogados.

## Objetivo
Optimizar la organización y control de la información legal mediante una solución tecnológica.

## Metodología
El proyecto se desarrolla bajo la metodología ágil Scrum, organizando el trabajo en sprints.
